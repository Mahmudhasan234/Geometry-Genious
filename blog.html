1. What is DOM and what is the purpose of DOM?

Document Object Model (DOM) is a interface for web document which allow program to interact with the document structure, style, and content.


2. How will you select HTML elements using DOM? Name the DOM methods.

JavaScript provides six methods which can be used to select HTML elemnet. The methods are following:
  • getElementById
  • getElementsByTagName
  • getElementsByClassName
  • getElementsByName
  • querySelector
  • querySelectorAll


3. What is event bubble?

Event Bubbling is a concept of DOM. It happened when an element receive an event. As an example, when in a click function triggered it receive an event and start bubbling up in the DOM tree until it reached to the root Element


4. What is a callback function and why will you use it?

When a function take a different function as perameter, that function which pass through is a callback function. Usually javacript runs in a sequential order. However sometimes there are some cases that code will run after something else happens which can not be sequentialed.Callback function ensure that the function will not run untill something happened.
